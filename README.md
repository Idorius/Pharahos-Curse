# Pharaoh's Curse

Projet de Roguelike Égyptien

## Description
Un jeu de type roguelike se déroulant dans l'Égypte ancienne, avec des thèmes pharaoniques et des mécaniques de jeu classiques du genre.

## Installation et Configuration

### Prérequis
- Git installé sur votre système
- Compte GitHub avec accès à l'organisation Idorius

### Configuration Git
1. Ouvrez une invite de commande ou PowerShell
2. Naviguez vers le dossier du projet :
   ```bash
   cd "C:\Users\eliot\Desktop\Idorius\Pharaoh's Curse"
   ```
3. Initialisez le repository Git :
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/Idorius/Pharahos-Curse.git
   git push -u origin main
   ```

### Publication sur GitHub
Le projet sera publié sur : https://github.com/Idorius/Pharahos-Curse

Assurez-vous d'être connecté avec le compte owner (Urioxi) pour pousser les modifications.

## Structure du Projet
- `config/` - Fichiers de configuration
- `README.md` - Ce fichier
- `.gitignore` - Fichiers à exclure du versioning
